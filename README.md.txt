# Garbage Classification Project

Proyek ini bertujuan untuk mengklasifikasikan gambar sampah ke dalam kategori tertentu menggunakan deep learning dengan TensorFlow/Keras. Model ini dapat membedakan antara kardus, kaca, logam, kertas, plastik, dan sampah lainnya.

## Daftar Kategori
- Cardboard
- Glass
- Metal
- Paper
- Plastic
- Trash

## Teknologi yang Digunakan
- **Bahasa Pemrograman**: Python
- **Library**:
  - TensorFlow/Keras
  - OpenCV
  - scikit-image
  - Pandas, NumPy
  - Matplotlib/Seaborn
- **Tools**: Google Colab, Kaggle API
